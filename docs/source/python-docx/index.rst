python-docx - Build Word Document in Python
==============================================================================

- GitHub: https://github.com/python-openxml/python-docx

Overview
------------------------------------------------------------------------------
这个库是用在 Python 中创建 Word 文档的. 它不支持对已有的 Word 文档进行编辑.
